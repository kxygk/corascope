Corascope

Free and opens source software for inspecting, cropping and stitching ITRAX XRF core scans

WARNING : The current software version only works for scans done at a 0.5mm/pixel resolution. The issue is being addressed..

  • Download A simple JAR excutable. No installation, just download and go. Requires Java 11 (or higher)
  • Quick Start A quick visual walk through loading, cropping and merging a core for export
  • Manual A indepth dive into the features available for core editing as well as details of how
  • Source All code is available on Github
  • Samples A couple of adjacent core scans are provided for testing:
    • A 0- 134cm - Optical, XRF Scan
    • B 134-275cm - Optical, XRF Scan
